# AUTOLIVE Setup Guide

## Backend
1. Install dependencies: `npm install`
2. Configure `.env` with MongoDB URL and API keys
3. Run server: `node backend/server.js`

## Frontend
1. Open `frontend/dashboard/index.html` in browser
2. Dummy data already included for demo

## AI
1. Predictor, Title Generator, Shorts Generator included in `ai/`

## Database
1. Use `database/schema.sql` to setup PostgreSQL database
