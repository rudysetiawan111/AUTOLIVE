# AUTOLIVE API

## Endpoints

### Channels
- GET /api/channels → list all channels (YouTube + TikTok)

### Videos
- GET /api/videos → list all videos
- POST /api/videos/upload → upload video (YouTube/TikTok)

### Viral
- GET /api/viral → list viral videos

### Workflow
- POST /api/workflow/trigger → trigger workflow

### Analytics
- GET /api/analytics → get engagement & categories

### Auth
- POST /api/auth/login → login user (dummy token)
